Peekaboo

Android style pull down menu, for mobile and non-mobile browsers

Basic Usage

To use simply define a your peekaboo menu in the following way, the library only expects a block element with a class of peekaboo and a block element with class handle inside: 

<div class="peekaboo">
  <div>
    <!-- Menu contents go here... -->
  </div>
  <div class="handle">
    <!-- Style this appropriately -->
  </div>
</div>

Configuration & Control

There are also a number of configuration items 

$(document).ready(function() {
  peekaboo.panel('#jqueryId').configure({
    'animation_speed': 3,    // Speed in milliseconds
    'initial_state': 'open', // Initial state onload
    'open_hook': function() {
      /* Hook called on open */
    },
    'close_hook': function() {
      /* Hook called when closed */
    }
  })
})

You can also control the menu programatically 

peekaboo.panel('#jqueryId').open();
peekaboo.panel('#jqueryId').close();
peekaboo.panel('#jqueryId').toggle();

The above will animate the panel between the open and close state. If you want to disable this pass true to any of these procs: 

// Amination disabled
peekaboo.panel('#jqueryId').open(true);
peekaboo.panel('#jqueryId').close(true);
peekaboo.panel('#jqueryId').toggle(true);
